What is Avien 5.5?

Avien 5.5 is a body-care collection built around named routines. The range combines everyday cleansing and moisturizing with targeted options you can spot instantly from the label: Glycolic Acid, Salicylic Acid, Vitamin C, Kojic Acid, Alpha Arbutin + Niacinamide, and Retinol + Collagen.

It was formulated in Maryland, USA, but manufactured in China.

General guidance: if you are new to exfoliating or retinol-based body care, patch test and introduce one "active" product at a time.

Avien 5.5 products (detailed guide)

Below is a clear, product-by-product guide to the Avien 5.5 range: what each one is used for, the highlighted ingredients on the label, and how to build a simple routine. For the complete ingredient list (INCI) and directions for your bottle, always read the label.

The Smoother Glycolic Acid Body Wash

Use for: rough texture, dull-looking skin, and a smoother feel.

  • Highlighted ingredient: Glycolic Acid (AHA).
  • Composition (high level): cleansing base + AHA + humectants/conditioners (see label for full INCI).
  • How to use: start 3–4x weekly, rinse well, then moisturize.
  • Routine tip: pair with The Smoother Body Lotion for a matched routine.

The Glow Getter Kojic Acid Body Wash

Use for: glow care and support for uneven-looking tone.

  • Highlighted ingredient: Kojic Acid.
  • Composition (high level): cleansing base + brightening-support ingredient + humectants (see label).
  • How to use: use daily or alternate days based on comfort; moisturize after shower.
  • Routine tip: pair with The Glow Getter Body Lotion.

The Purifier Alpha Arbutin + Niacinamide Body Wash

Use for: the look of marks, uneven tone, and overall skin clarity.

  • Highlighted ingredients: Alpha Arbutin + Niacinamide.
  • Composition (high level): cleansing base + targeted brightening/support ingredients + conditioners (see label).
  • How to use: use daily or alternate days; keep routine consistent for weeks.
  • Routine tip: pair with The Purifier Body Lotion.

The Wonder Body Wash

Use for: simple, everyday cleansing.

  • Highlighted ingredient: everyday routine (no single active highlighted in the name).
  • Composition (high level): cleansing base + skin-conditioning agents (see label).
  • How to use: suitable for daily showers; moisturize after use.
  • Routine tip: works with any Avien 5.5 body lotion.

The Perfector Salicylic Acid Body Wash

Use for: clarifying body acne-prone areas and clogged pores.

  • Highlighted ingredient: Salicylic Acid (BHA).
  • Composition (high level): cleansing base + BHA + comfort agents (see label).
  • How to use: use 3–5x weekly on needed areas; rinse well.
  • Routine tip: keep other exfoliating products minimal on the same day.

The Restorer Retinol + Collagen Body Wash

Use for: renewal-focused cleansing and smoother-looking texture.

  • Highlighted ingredients: Retinol + Collagen (routine name).
  • Composition (high level): cleansing base + renewal-positioned ingredients (see label).
  • How to use: start 2–3x weekly; moisturize after shower; use SPF in daytime.
  • Note: if pregnant or breastfeeding, consult a clinician before using retinoid products.

The Brightener Vitamin C Body Wash

Use for: brightening cleanse and glow-focused routines.

  • Highlighted ingredient: Vitamin C (antioxidant support).
  • Composition (high level): cleansing base + vitamin C routine positioning + conditioners (see label).
  • How to use: use daily or alternate days; follow with a lotion.
  • Routine tip: pair with The Brightener Vitamin C Body Lotion.

The Purifier Body Lotion

Use for: daily moisture + support for an even-looking tone.

  • Highlighted ingredients: Alpha Arbutin + Niacinamide (routine family).
  • Composition (high level): emollients + humectants + routine-positioned ingredients (see label).
  • How to use: apply after shower; focus on areas of concern daily.
  • Routine tip: pair with The Purifier body wash.

The Glow Getter Body Lotion

Use for: glow care + daily moisturizing.

  • Highlighted ingredient: Kojic Acid (routine family).
  • Composition (high level): moisturizing base + routine-positioned ingredients (see label).
  • How to use: apply daily after shower; be consistent.
  • Routine tip: pair with The Glow Getter body wash.

The Wonder Body Lotion
